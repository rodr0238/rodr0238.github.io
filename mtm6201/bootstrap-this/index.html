ABOUT BOOTSTRAP

Bootstrap is an open source front end development framework to create websites and apps. The Bootstrap framework is built on HTML, CSS, and JavaScript. Bootstrap facilitates the development of responsive, mobile-first sites and apps. Bootstrap includes user interface components, layouts and JavaScript tools along with the framework for implementation.

BACKGROUNDS

There are different classes for various background colors.

COLORS: bg-primary, bg-secondary, bg-success, .bg-danger, .bg-warning, bg-info, bg-light, bg-dark, etc.

DISPLAY

.d-{value} classes are used to apply the display property to elements.

.d-{value} is used for xs breakpoint while .d-{breakpoint}-{value} is used for other breakpoints such as sm, md, lg, and xl.

The values can be any one of the these - none, inline, inline-block, block, table, table-cell, table-row, flex and inline-flex.

Screen widths can be effected by the media queries with the given breakpoint or larger.

Examples

display: inline;
display: block;
display: block;
display: flex;

LAYOUT

Containers

Containers are required when using the default grid system. Containers are used to contain, pad and sometimes center the content inside it. There are 3 different containers in Bootstrap.
[.container]: This container sets a max-width according to the breakpoint.
[.container-fluid]: This container takes 100% width at all breakpoints.
[.container-sm, .container-md, .container-lg, .container-xl, .container-xxl]: This container allow to specify a class that is 100% of width untill the specified breakpoint is reached.

COMPONENTS

Navigation Bar

Nav tags are applied to a class of .navbar for default navbar and .navbar-expand + {-sm/-md/-lg/-xl/-xxl} for responsive collapsing and color scheme classes.
Spacing and Flex utility clsses can be used to controll spacing and alignment within the navbars.
Navbars are responsive by default.

CARDS

Card is a flexible and extensible container of content. It includes header, footer, background colors, and many display options.

We can nest cards in rows and use the grid system to control multiple cards. In addition, we can style the cards as we like using Bootstrap utilities as well as Custom CSS.

We can even add navigation inside cards using Bootstrap's nav components.

Multiple cards can be grouped together with the help of Card layouts.

It is also easy to change up the color schemes and text alignment and we can even play around with the images.
